# Game Ranking Web App

This project is a web application built with Angular that allows users to log in and track their game statistics such as highest score and total play time.  
It is designed to integrate with a Phaser-based game and act as a simple ranking and profile system.

## Purpose of this repository

This repository was created as a portfolio project to showcase skills in:

- Frontend development with Angular.
- Integration with external services (Firebase).
- Authentication and user management.
- Data persistence using a cloud database.
- Basic UI design with Angular Material.
- Clean project structure and modular architecture.

The main goal is to demonstrate how a modern web application can interact with a game and provide features such as ranking, user profiles, and persistent data.

## Features

- User registration and login.
- Save and display highest score.
- Track total play time.
- Ranking table of players.
- Responsive UI.
- Real-time database updates.

## Tech Stack

- Angular 21
- Firebase (Auth + Firestore)
- TypeScript
- Angular Material
- HTML / CSS
